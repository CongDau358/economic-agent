# Memory Lifecycle Rules

## Purpose
Define how knowledge enters, evolves, and expires.

## Lifecycle Stages

### 1. Ingestion
Document added to system.

### 2. Processing
- extraction
- chunking
- embedding
- metadata tagging

### 3. Retrieval
Document becomes searchable.

### 4. Validation
Evidence quality evaluated.

### 5. Archival
Outdated information downgraded in retrieval ranking.
